1    Symmetric Encryption                                                  (40 marks)The Feistel structure Figure 1 is a foundation of many modern symmetric block ciphers.  It requires a 2w bit input and splits it into a left (bottom) and right (top) part of w bit each.  It then proceeds in n rounds to produce a 2w bit output.  Each round consists of the following steps: ‚Ä¢  The top w bits are combined with the round key ki  by a round function f. ‚Ä¢  The bottom w bits are combined with the output of f by bit-wise exclusive or (XOR). ‚Ä¢  Then, the top and bottom bits are swapped and passed on to the next round.
